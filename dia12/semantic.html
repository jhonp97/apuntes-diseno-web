Ejemplos de etiquetas no semanticas: div, span
Ejemplos de etiquetas semanticas: table, form, article

Header

Es el encabezado de una pagina, seccion o articulo. Incluye elementos como el logo, texto introductorio, menu de navegacion, etc.Puede haber varios elementos header en una misma pagina, pero no puede haber un header dentro de un footer, o de otro header.

Footer

Es el pie de pagina de un sitio o seccion, puede haber miultiples por pagina, puede incluir informacion como:
  • Informacion de contacto
  • Enlaces a las RRSS
  • Mapa del sitio
  • Creditos
  • Formlario de contacto

Nav

Define un conjunto de enlaces de navegacion, no todos los links deben estar dentro de un "Nav". por ejemplo, un menu de navegacion principal, un menu de navegacion secundario, un menu de navegacion de paginacion, etc.

Main

Crea un contenedor para el contenido principal de nuestra pagina,Este debe ser unico y no debe incluir contenido que sea repetitivo en otras secciones, como por ejemplo del sidebar, logos, etc. No puede ser hijo de etiquetas como header, footer, nav, aside, article.

Aside

Define el contenido relacionado indirectamente con el contenido principal de la pagina, como por ejemplo, un sidebar, publicidad, etc. Elementos que solemos poner en un aside:
  • Publicidad
  • Biografia del autor
  • Enlaces relacionados
  • Informacion de perfil

Section

Define una seccion en una pagina donde se grupan los contenidos relacionados, y normalmente incluye un titulo. Puede haber varias secciones en una misma pagina, pero no puede haber una seccion dentro de otra seccion.
Una web se compone de varias secciones cada una con su contenido y proposito.
  • Capitulos
  • Introduccion
  • Noticias
  • Informacion de contacto

Article

Define un contenido INDEPENDIENTE. El midmo puede ser compartido o reutilizado en otro sitio sin requerir contexto y sin perder su significado.
Tambien suele incluir un titulo
Ejemplos de uso:
  • Un post de un blog
  • Un articulo de una revista
  • Un comentario se usuario (rese√±a)
  • Un widget o tarjeta de producto

    • Puedo encontrar articulos dentro de secciones, y secciones dentro de articulos. Un articulo puede tener multiples secciones. Una seecion puede tener multiples articulos.

    Figure

    Define contenido independiente, como imagenes, graficos, ilustraciones, diagramas, codigo, etc.
    Tambien se puede usar para agrupar contenido relacionado, como una imagen con su titulo y descripcion.
    Dentro de ella se suele utilizar la etiquetas figcaption para describir el contenido, esto es ademas del ya incluido "alt" en la etiqueta img.

    Mark

    Define un texto resaltado o marcado, es decir, un texto que se quiere destacar del resto. Normalente se muestra con texto amarillo
