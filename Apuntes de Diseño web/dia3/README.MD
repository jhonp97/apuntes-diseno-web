selectores CSS
Para seleccionar un grupo de elementos podemos usar el atributo de clase y el selecetor "."

Para seleccionar un unico item podemos usar el atributo de id, y el selector "#"

``` css
div{
color:blue;
}

.alumnos{
color: red;
}

#juancito{
color: green;
}
```
```html
<di class="alumnos" id="juancito">hola</div>

```
